# G20 LC Bonds Dashboard v2.1

Cyber Noir data visualization dashboard for G20 local currency bond markets, BRICS + LATAM debt composition, currency volatility, and institutional infrastructure analysis.

## Stack

- React 19 + TypeScript + Vite
- Tailwind CSS (Cyber Noir palette)
- Recharts (data visualization)
- html2canvas + jsPDF (export PNG/PDF)

## Features

- **8 interactive sections**: Hero KPIs, Brazil Spotlight, Market Size, Spreads vs USD, Volatility Ranking, TCX Hedging, Debt Composition, Stability Scatter
- **PT/EN toggle** with ZH-ready architecture
- **Export PNG/PDF/JSON** per chart
- **Source link overlays** with methodology
- **Embed widgets** per section (iframe snippets)
- **Region filter**: All / BRICS / LATAM
- **LIVE BRL/USD Ptax** badge
- **⚠️ est. badges** on estimated data
- **Scroll-reveal animations**

## Data Sources

BIS, IMF WEO, CEPAL, Bloomberg, BCB, PBOC, NDB, TCX, CIPS
